KFC Drive-Thru Automation - Installation Instructions 
 
INSTALLATION: 
1. Run Install.bat as Administrator 
2. Or simply double-click KFC_DriveThru_Automation.exe to run directly 
 
USAGE: 
- Double-click the desktop shortcut 
- Click "Start Automation" button 
- Wait for completion 
 
REQUIREMENTS: 
- Windows 10 or later 
- Microsoft Excel installed 
- Internet connection
